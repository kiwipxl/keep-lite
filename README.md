## Development
Follow these instructions to setup your local development environment:

## Client
Run `npm install && npm run dev`.

## Server
* Download and install postgresql. Create a database with the following properties:
User: postgres
Host: database
Password: postgres
Port: 5432

* Run `npm install && npm run dev`
